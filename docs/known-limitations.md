# 알려진 제한사항

## 현재 미구현
- Search
- Thumbnail
- Bookmark
- PDFium
- Pan
- Double Tap Zoom
- Swipe Page Navigation
- Multi Page Scroll
- PDF 편집
- 메모
- 형광펜
- OCR
- 클라우드 동기화

## 현재 제한
- Zoom은 scale 조정만 지원한다.
- Pan은 지원하지 않는다.
- 검색은 지원하지 않는다.
- 썸네일은 지원하지 않는다.
- 매우 큰 PDF는 기기 성능에 영향을 줄 수 있다.
- 일부 PDF는 Android PdfRenderer 한계로 렌더링 실패 가능성이 있다.
