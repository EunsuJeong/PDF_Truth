# 릴리즈 후보 체크리스트

## 1. 빌드 기준
- [ ] `gradlew.bat assembleDebug`가 성공한다.
- [ ] APK 파일명이 `PDF_YYMMDD_NUMBER.apk` 규칙을 따른다.
- [ ] 기존 APK를 덮어쓰지 않는다.
- [ ] GitHub Actions CI가 성공한다.

## 2. 정책 기준
- [ ] 무료 제공 정책을 유지한다.
- [ ] 광고가 없다.
- [ ] 로그인 기능이 없다.
- [ ] 구독 기능이 없다.
- [ ] 개인정보 수집이 없다.
- [ ] 외부 서버 의존 기능이 없다.
- [ ] 오프라인 중심 동작을 유지한다.

## 3. 기능 기준
- [ ] PDF 선택 기능이 동작한다.
- [ ] PDF 렌더링 기능이 동작한다.
- [ ] 페이지 이동 기능이 동작한다.
- [ ] 마지막 페이지 복원 기능이 동작한다.
- [ ] 최근 문서 기능이 동작한다.
- [ ] 제한적 Zoom 기능이 동작한다.

## 4. 안정성 기준
- [ ] 권한 만료 안내가 정상 표시된다.
- [ ] 파일 삭제 안내가 정상 표시된다.
- [ ] 손상 PDF 안내가 정상 표시된다.
- [ ] render race condition 방지가 동작한다.
- [ ] Bitmap recycle 처리가 정상 동작한다.

## 5. 배포 보류 기준
아래 항목 중 하나라도 발생하면 배포를 보류한다.
- [ ] 앱 크래시 발생
- [ ] PDF 렌더링 깨짐
- [ ] 페이지 이동 오류
- [ ] 최근 문서 오류
- [ ] 마지막 페이지 저장/복원 오류
- [ ] APK 파일명 규칙 오류
