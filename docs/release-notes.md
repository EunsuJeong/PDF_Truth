# PDF Truth RC1 릴리즈 노트

## 릴리즈 정보
- 이름: PDF Truth RC1
- 태그: v0.1.0-rc1
- 기준 브랜치: main
- 기준 APK: PDF_260525_02.apk

## 현재 안정 기능
- PDF 파일 선택 및 열기
- 단일 페이지 표시
- 좌우 스와이프 페이지 이동
- 부드러운 pinch zoom
- 최근 문서 목록
- 마지막 페이지 저장 및 복원

## 미구현 기능
- Search
- Thumbnail
- Bookmark
- PDF 편집
- 필기/형광펜

## 알려진 제한사항
- 현재 zoom은 조작감 우선으로 동작하며, 확대 시 고해상도 재렌더링은 적용하지 않음
- 다중 페이지 뷰는 지원하지 않음
- 클라우드 동기화 및 계정 기능은 제공하지 않음
